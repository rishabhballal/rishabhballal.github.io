Geometric structures

The simplest mathematical structure is a set. Addition of certain axioms at each stage elevate the set into more useful geometric structures. Once it is equipped with a topology, it can be extended to a topological manifold.


Below are some standard 2-dimensional manifolds that are topological embeddings in the familiar 3-dimensional Euclidean space R3. For the algorithm behind these models, refer this minimal working example of a torus.
