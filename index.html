Constructing 3D models using canvas in JavaScript

Since there are multiple models on this site, object-oriented programming eliminates repetitive code. That particular approach can be found in the models.js file on the Github repository for this site. Given here, for simplicity, is a sequential breakdown of a single model - the 2-torus.


Canvas initialisation

The Canvas API is used to create graphics in JavaScript and render them in HTML. Lines 5-6 centre the canvas origin and invert the Y-axis, so as to work in the usual Cartesian coordinate system.



Coordinates

From its mathematical identification with the direct product of two 1-spheres (or by simple visual intuition), the 2-torus needs only two angles phi and theta to parameterize its coordinates. Conversion of the coordinates from spherical to Cartesian is given by the equations in lines 12-14.



Orientation

Rotating the model can provide a better view. This is done by multiplying each coordinate with the chosen rotation matrix. Although not immediately obvious, the general rotation equations given in lines 5-7 result from the cyclic nature of Cartesian coordinates.



Stereographic projection

In the final step, it is necessary to plot a 3-dimensional coordinates on a 2-dimensional screen. Projection eliminates the third coordinate, but not before encoding its information in the first two coordinates.


As can be seen in the equations of stereographic projection on lines 3-4, the projected coordinates are proportional to the first two coordinates, scaled by a factor which depends on the third. The variable dist represents the distance, in the direction of the third coordinate, from which one is observing the model. If it is too small, the projection will diverge to infinity, and if it is too large, it will not lend the perspective of depth. The remaining lines of code exhibit simple use of the canvas context in drawing lines that connect the coordinates.


Compiling the above four snippets of code will yield the 2-torus model from the previous page, while the other models can be recreated with an appropriate modification of the coordinate equations.
